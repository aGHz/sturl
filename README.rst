Install
-------

You can install sturl from PyPI with `pip <http://www.pip-installer.org/>`_:

.. code-block:: sh

    $ pip install sturl

or from `GitHub <https://github.com/aGHz/sturl>`_ with git:

.. code-block:: sh

    $ git clone https://github.com/aGHz/sturl.git
    $ cd sturl && python setup.py install
